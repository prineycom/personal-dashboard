# Personal Dashboard

ИИ-оркестратор личной продуктивности.
Hermes-агент управляет задачами, фокус-целями, финансами, здоровьем, трекерами и временем через унифицированные интеграции.

## Документация

Дизайн-документы и планирование хранятся в vault:
`priney-memory/40-Projects/personal-dashboard/`

## Архитектура

```
Hermes Agent → MCP / API → Сервисы (Supabase, Toggl, TickTick, ZenMoney)
```

## Структура репозитория

- `mcp/` — MCP серверы для внешних сервисов
- `schema/` — SQL схемы и миграции
- `scripts/` — утилиты (бекапы, синхронизация)
- `docs/` — сгенерированная документация (публичная)
